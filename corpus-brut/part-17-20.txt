Libya.
D anwa-t Saɛid Sifaw ?
Saɛid Sifaw, neɣ Lmeḥruq, d amedyaz Amaziɣ n Libya.
Ilul deg 1946, di Agadu, deg yirebbi n yidurar n Neffusa, anẓul-amalu n tmurt n Libya n wass-a.
Yella yettwassen abeɛda s yisefra-is, timmeɣnest-is d umennuɣ-is imegzi mgal adabu amhersan n Muɛemmer Lqeddafi, imi ɣur yal Amaziɣ n Libya, ad yeqqim d azamul d timmuzɣa d uzbu di temnaḍt-a n Tmazɣa.
Ugar n tẓuri n wawal, yeɣra Saɛid Sifaw tasnujjya di tmurt n Maser.
Maca, ɣef tmentilt n yimagraden i yettaru imir-nni, tettwakkes-as teggazt-is.
D ayen i t-iḥettmen ad yuɣal ɣer Trablus deg 1967, ad iɣer azref, anda i d-iḥella agerdas n ubugaṭu.
Deg useggas 1979, yeslek Saɛid Sifaw seg tmettant deg yiwet n txazabit.
Yeḍra-as yiwen n usehwu (accident) n ubrid d amihan, i as-d-wwin ssrabes n udabu n Libya, ɣef wawal amatu i yuzzlen.
Seg usehwu-a, tenṭeḍ tɛuḍert amedyaz, armi ikemmel tudert-is d ameɛḍur n tfekka.
D ayen i t-yeǧǧan, seg tikkelt ɣer tayeḍ, yettdawi berra tmurt-is.
Seg wayen i yura, ad naf yegmer-d ddeqs n tmucuha d teqsiḍin n tgemmi n tmurt-is, am wakken yura ddeqs n yisefra s tmaziɣt n Libya d teɛrabt.
Di tudert-is, yessuffeɣ-d sin n yidlisin, « Taɣect n tzeggant n yiḍ », d ammud n tmucuha i yettwasuqlen ɣer teɛrabt, akked wammud n tmedyazt « Isefra isusamen ».
Seg umezruy, ass n 15 yunyu 2009, ugin yimḍebbren n tmurt n Merruk ad d-yettwasken i tɣamsa Udlis aberkan n Saɛid Sifaw deg Uqusis n Tɣamsa n Rribaṭ.
Seg wayen i d-yennulfan ɣef yiman n Saɛid Sifaw, yella yiwen n warraz iɣef ur nesɛi ddeqs n yisallen.
Isem-is warraz-a « Arraz Saɛid Sifaw », yettunefk i Muḥemmed Farid Zalhud deg useggas n 1997.
Seg yisefra i yura, ad d-nebder kan “Amezruy-nneɣ”, “Tidet”, “Tayri”, “Idles-nwen” akked “Amussnaw”.
Sin-a ineggura dɣa, yecna-ten Ferḥat Imaziɣen Imula deg tezlatin i yettwassnen ayendin.
Yemmut Saɛid Sifaw deg useggas n 1994, di Ǧerba, tamurt n Tunes.
